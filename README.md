# Calculator Tkinter

Une application de calculatrice simple et intuitive développée avec **Python** et la bibliothèque **Tkinter**. Ce projet permet d'effectuer des opérations arithmétiques de base (addition, soustraction, multiplication, division) via une interface graphique moderne.

## 🚀 Fonctionnalités
- **Interface Graphique :** Une GUI propre et réactive créée avec Tkinter.
- **Opérations de base :** Addition, soustraction, multiplication et division.
- **Gestion des erreurs :** Prise en charge des erreurs de calcul (ex: division par zéro).
- **Code épuré :** Structure de code facile à lire et à modifier.

## 🛠 Prérequis
Pour exécuter ce projet, vous devez avoir **Python** installé sur votre machine.
