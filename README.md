# Calculator Tkinter

Une application de calculatrice simple et intuitive développée avec **Python** et la bibliothèque **Tkinter**.

## Fonctionnalités
- Effectuer des calculs arithmétiques de base (addition, soustraction, multiplication, division).
- Interface graphique utilisateur (GUI) réactive.
- Code source propre et facile à comprendre pour les débutants.

## Prérequis
Pour exécuter ce projet, vous devez avoir **Python** installé sur votre machine.
